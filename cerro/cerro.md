# Initial (first) install

1. Download the ROM
2. Download the recovery
3. Unlock the bootloader
4. Flash recovery through fastboot or edl
5. Boot into recovery
5. Wipe data
6. Install the ROM using sideload
7. Reboot
8. Done

# Updating (OTA)

1. Check for updates in YAAP Updater
2. Download the update
3. Update
4. Reboot
5. Done

OR

1. Download the ROM update
2. Enter recovery
3. Flash using adb sideload
4. Reboot
5. Done
